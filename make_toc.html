Beispieldokumente

Wir nehmen einige der Beispieldokumente aus dem Tutorium und fügen sie auf verschiedene Arten zu einem Dokument zusammen.

Einfaches Aneinanderhängen

Drei Dokumente einfach aneinanderhängen.

file1 <- "t1_table_1/t1_table_one.docx"
file2 <- "t1_table_1/t1_table_one_mit_total.docx"
file3 <- "t3_lm/t3_table_lm.docx"
file_out <- "main_doc.docx"

x <- read_docx(path = file1)
x <- body_add_break(x)
x <- body_add_docx(x, src = file2)
x <- body_add_break(x)
x <- body_add_docx(x, src = file3)
print(x, target = file_out)

Erster TOC-Versuch

TOC als Platzhalter einfügen und drei Dokumente aneinanderhängen.

file1 <- "t1_table_1/t1_table_one.docx"
file2 <- "t1_table_1/t1_table_one_mit_total.docx"
file3 <- "t3_lm/t3_table_lm.docx"
file_out <- "main_doc_toc_1.docx"

x <- read_docx()
x <- body_add_toc(x)
x <- body_add_break(x)
x <- body_add_docx(x, src = file1)
x <- body_add_break(x)
x <- body_add_docx(x, src = file2)
x <- body_add_break(x)
x <- body_add_docx(x, src = file3)
print(x, target = file_out)

Im Ergebnisdokument ist ein Platzhalter für das Inhaltsverzeichnis vorgesehen. Es kann aber noch nicht erstellt werden, da keine Formatvorlagen zugefügt worden sind.

Zweiter TOC-Versuch

TOC als Platzhalter einfügen und drei Dokumente aneinanderhängen.

Die Dokumente sind durch Paragraphen mit Überschriften und zugewiesenen Formatvorlagen getrennt.

file1 <- "t1_table_1/t1_table_one.docx"
file2 <- "t1_table_1/t1_table_one_mit_total.docx"
file3 <- "t3_lm/t3_table_lm.docx"
file_out <- "main_doc_toc_2.docx"

x <- read_docx()
x <- body_add_par(x, value = "Table of contents", style = "heading 1")
x <- body_add_toc(x)
x <- body_add_break(x)
x <- body_add_par(x, value = "Tables", style = "heading 1")
x <- body_add_par(x, value = "Table 1", style = "heading 2")
x <- body_add_docx(x, src = file1)
x <- body_add_break(x)
x <- body_add_par(x, value = "Table 2", style = "heading 2")
x <- body_add_docx(x, src = file2)
x <- body_add_break(x)
x <- body_add_docx(x, src = file3)
x <- body_add_par(x, value = "Table 3", style = "heading 2")
print(x, target = file_out)

Das sieht schon besser aus.

Dritter TOC-Versuch

Das vorige Dokument wird als Vorlage benutzt. Die Formatvorlagen für heading 1 und heading 2 werden angepasst, der Inhalt wird gelöscht und das Dokument wird als Vorlage gespeichert.

file1 <- "t1_table_1/t1_table_one.docx"
file2 <- "t1_table_1/t1_table_one_mit_total.docx"
file3 <- "t3_lm/t3_table_lm.docx"
file_out <- "main_doc_toc_3.docx"

x <- read_docx(path = "main_doc_toc_vorlage.docx")
x <- body_add_par(x, value = "Table of contents", style = "heading 1")
x <- body_add_toc(x)
x <- body_add_break(x)
x <- body_add_par(x, value = "Tables", style = "heading 1")
x <- body_add_par(x, value = "Table 1", style = "heading 2")
x <- body_add_docx(x, src = file1)
x <- body_add_break(x)
x <- body_add_par(x, value = "Table 2", style = "heading 2")
x <- body_add_docx(x, src = file2)
x <- body_add_break(x)
x <- body_add_docx(x, src = file3)
x <- body_add_par(x, value = "Table 3", style = "heading 2")
print(x, target = file_out)

Jetzt kommt das Inhaltsverzeichnis auf die Zielgerade.

Wenn ein Inhaltsverzeichnis in dieser Form und mit Überschriften erstellt wird, kann es sein, dass die einzelnen Tabellenprogramme angepasst werden müssen, damit es mit den Überschriften keine redundaten Informationen gibt.

Das hängt von den persönlichen Vorlieben und dienstlichen Vorgaben ab.
