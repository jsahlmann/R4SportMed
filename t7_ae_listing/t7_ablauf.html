Speichern als Word-Dokument

Mit dem Paket officer kann man ein flextable-Objekt als Worddokument speichern.

Wenn wir uns hier auf das autofit() verlassen, stellen wir fest, dass die Breite der Tabelle A4 quer überschreitet.

my_doc <- read_docx("table_vorlage_ae_listing.docx") %>% body_add_flextable(ft_ae)
print(my_doc, target = "t7_ae_listing_zu_breit.docx")

Wir müssen die Spaltenbreiten noch händisch anpassen und die Schriftgröße verringern.

ft_ae <- ft_ae %>% 
  fontsize(size = 8, part = "all") %>%
  width(j = 1, width = 1.5, unit = "cm") %>%
  width(j = 2, width = 1.2, unit = "cm") %>%
  width(j = c(3, 4, 8), width = 3.3, unit = "cm") %>%
  width(j = c(5), width = 2, unit = "cm") %>%
