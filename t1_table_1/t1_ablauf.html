Kategorielle Variablen auszählen

Das Geschlecht wird als kategorielle Variable erfasst und stratifiziert nach Behandlungsarm ausgezählt.

Die Anzahl der Studienteilnehmer wird bestimmt und als Basis für die Berechnung des Prozentanteils benutzt.

Die absolute und die relative Häufigkeit werden als String zusammengefasst und in der Variablen value gespeichert.

Kontinuierliche Variable Alter beschreiben

Das Alter liegt als kontinuierliche Variable vor.

Die deskriptiven Statistiken werden nach Behandlungsarm stratifiziert berechnet.

Hier werden beispielhaft Mittelwert, Standardabweichung, Median und Anzahl der Werte berechnet.

df2 <- df %>% group_by(treatment) %>%
  summarize(m = mean(age),
            s = sd(age),
            md = median(age),
            n = n()) %>%
  ungroup()
df2$m <- as.character(round(df2$m, 2))
df2$s <- as.character(round(df2$s, 3))
df2$ms <- paste0(df2$m, " \U00B1 ", df2$s)
df2$md <- as.character(df2$md)
df2$n <- as.character(df2$n)
